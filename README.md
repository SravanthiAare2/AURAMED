# 🏥  AURAMED: An Explainable AI system for Personalized Drug & Dosage Recommendation

## 📌 Project Overview
AURAMED is an AI-based healthcare application that helps in drug recommendation, dosage prediction, and ADR (Adverse Drug Reaction) risk analysis.  
It uses Machine Learning to support safer and faster clinical decisions.

## ⚙️ Technologies Used
- Python  
- Streamlit  
- Pandas  
- NumPy  
- Scikit-learn  
- SHAP  
- Matplotlib  
- ReportLab  

## ▶ How to Run the Project

### 1. Install required libraries
pip install -r requirements.txt

### 2. Run the application
streamlit run app.py

## 📂 Project Files
- `app.py` – Main application file  
- `auramed_clinicaldataset.xlsx` – Dataset  
- `requirements.txt` – Required libraries
